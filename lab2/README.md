# Лабораторная работа №2: ETL на Apache Spark
В данной лабораторной был реализован ETL-пайплайн, который перекладывает сырые данные о продажах из PostgreSQL в БД ClickHouse с помощью Apache Spark.

В процессе плоские данные преобразуются в модель "Снежинка", а затем на их основе строятся 6 витрин данных.

## Структура проекта
- docker-compose.yml - поднимает Postgres, ClickHouse и Spark (Jupyter).
- 01_init.sql - скрипт инициализации Postgres. При первом старте он создает схему базы данных и заливает туда 10 000 строк из csv-файлов.
- etl/ - папка с Jupyter-ноутбуком, где написана логика Spark-джобов.

## Как запустить Spark-джобы

1. Поднять контейнеры:
   ```bash
   make up
   ```

2. Открыть Jupyter Notebook:
- Перейти в браузере на `http://localhost:8888`
- Ввести токен `lab2`
- Открыть ноутбук `work/etl/notebook.ipynb`

3. Запустить ячейки:

    Шаг 1: Установка драйверов и подключение к БД.

    Шаг 2: Выгрузка сырых данных из mock_data в Postgres.

    Шаг 3: Нарезка данных на измерения (dim_brands, dim_customers, ...) и формирование таблицы фактов fact_sales.

    Шаг 4: Сохранение готовой модели обратно в Postgres.

    Шаг 5: Расчет 6 витрин с помощью Spark (по продуктам, клиентам, времени и т.д.).

    Шаг 6: Выгрузка готовых витрин в ClickHouse.
